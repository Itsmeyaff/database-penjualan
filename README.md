# Database Penjualan Toko Elektronik

## Deskripsi

Proyek database sederhana untuk mengelola data penjualan toko elektronik.

## Fitur

- Manajemen Produk
- Manajemen Pelanggan
- Transaksi Penjualan
- Detail Transaksi

## Tools

- SQLite3
- Draw.io (ERD)

## Struktur Database

- produk
- pelanggan
- transaksi
- detail_transaksi
